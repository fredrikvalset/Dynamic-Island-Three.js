# Dynamic Island-Three.js

[Link to live site](https://three.freval.no)


## Controls 

### Mouse

- Orbit: left mouse 
- Zoom: middle mouse, or mousewheel 
- Pan: right mouse, or left mouse + ctrl/meta/shiftKey 

### Touch

- Orbit: one-finger move
- Zoom: two-finger spread or squish
- Pan: two-finger move
